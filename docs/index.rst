utract-mcp Documentation
========================

**utract-mcp** is a Model Context Protocol (MCP) server for `Plane <https://plane.so>`_ /
`utrack <https://utrack.bht.co.id>`_ project management platforms.
It exposes 44+ tools for managing projects, work items, sprints, modules, states, labels,
members, comments, and activity logs through any MCP-compatible client.
